# Focus AI Todo

## Overview
Focus AI Todo is a productivity application designed to help users manage their tasks effectively. It employs artificial intelligence to prioritize and suggest tasks based on user preferences and workload.

## Features
- **Task Management:** Add, edit, and delete tasks easily.
- **AI Suggestions:** Use AI to prioritize your tasks smartly.
- **User-Friendly Interface:** Simple and intuitive design for enhanced user experience.
- **Cross-Platform Access:** Available on multiple devices to keep you productive anywhere.

## Getting Started
1. Clone the repository: `git clone https://github.com/woshixiyangyang/focus_ai_todo.git`
2. Navigate into the project directory: `cd focus_ai_todo`
3. Install dependencies: `npm install`
4. Start the application: `npm start`
